
```bash

npm run dev
```

## Troubleshooting

**Port already in use?**
- Backend: Change PORT in `backend/server.js`
- Frontend: Change port in `frontend/vite.config.js`

**Database errors?**
```bash
cd backend
rm ecommerce.db
npm run init-db
```

**Module not found?**
```bash
# Run in both directories:
npm install
```
